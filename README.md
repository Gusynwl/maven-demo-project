# Maven 脚手架

主要包含以下 Maven 标准目录结构和文件，以及一些常用插件、功能等。

```
- src
  - main
    - java
    - resources
    - filters
	  - filter-dev.properties
	  - filter-test.properties
	  - filter-mirror.properties
	  - filter-prod.properties
    - webapp
  - test
    - java
    - resources
- target
- pom.xml
```

可用于快速创建 Maven 项目。
